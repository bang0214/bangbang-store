## 简要说明
此仓库为后端代码且集成了管理端前端代码。

项目由一个父模块和两个子模块组成，common模块中存储要用到的实体类pojo，传递参数param，前端展示实体vo和工具类utils等。

server模块中存储主要业务逻辑代码（mapper、service、controller）和一些配置类，工具类；resources下存储了前端要访问的一些静态图片资源（public），管理端前端的页面文件（static，templates）。
